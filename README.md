# Кузнечик

Блочный шифр ГОСТ Р 34.12-2015 на языке Go

## Описание

Реализация блочного шифра Кузнечик (ГОСТ Р 34.12-2015) на языке Go. Криптографическая утилита для шифрования и расшифрования файлов.

Данная программа выполнена в рамках практического задания учебного курса "Криптографические методы защиты информации" студент Самарин Евгений.

## Структура проекта

```
.
├── main.go          # CLI и точки входа
├── core.go          # L/S-преобразования
├── keygen.go        # Генерация ключей
├── io.go            # I/O операции
├── defs.go          # Типы и константы
├── arithmetic.go    # Арифметика
├── encrypt.go       # Функции шифрования
├── decrypt.go       # Функции расшифрования
```

## Сборка

```bash
go build -o kuznechik
```

## Использование

### Шифрование файла

```bash
./kuznechik -e -i <input> -o <output> -k <key>
```

### Расшифрование файла

```bash
./kuznechik -d -i <input> -o <output> -k <key>
```

## Параметры

- `-e` — шифрование
- `-d` — расшифровка
- `-i <file>` — входной файл
- `-o <file>` — выходной файл
- `-k <key>` — ключ (64 hex символа)
