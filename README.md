# Кузнечик

Блочный шифр ГОСТ 28147-89 на языке Go

## Описание

Реализация блочного шифра Кузнечик (ГОСТ 28147-89) на языке Go. Криптографическая утилита для шифрования и расшифрования файлов.

Данная программа выполнена в рамках практического задания учебного курса "Криптографические методы защиты информации" студент Самарин Евгений.

## Структура проекта

```
.
├── main.go          # CLI и точки входа
├── core.go          # L-преобразование и вспомогательные таблицы
├── keygen.go        # Генерация ключей
├── io.go            # I/O операции
├── types.go         # Типы данных
├── constants.go     # Константы
├── arithmetic.go    # Арифметика
└── encrypt.go       # Функции шифрования
```

## Сборка

```bash
go build -o kuznechik
```

## Использование

### Шифрование файла

```bash
./kuznechik -e -i <input> -o <output> -k <key>
```

### Расшифрование файла

```bash
./kuznechik -d -i <input> -o <output> -k <key>
```

## Параметры

- `-e` — шифрование
- `-d` — расшифровка
- `-i <file>` — входной файл
- `-o <file>` — выходной файл
- `-k <key>` — ключ (64 hex символа)
